# FA v3 Architecture Index

This directory is the canonical architecture set for FA v3.

## Read Order
1. `00_status.md`
2. `actual_state/01_implementation_baseline.md`
3. `actual_state/02_phase_status_reconciliation.md`
4. `target_state/02_layer_model.md`
5. `target_state/07_phase_roadmap_from_now.md`
6. `governance/02_canonical_sources.md`

## Folder Layout
- `actual_state/`: code-verified current implementation.
- `target_state/`: FA v3 design target and delivery plan.
- `governance/`: source-of-truth, lifecycle, and sync controls.

## Scope Note
FA v3 intentionally separates implemented architecture from planned architecture to prevent v9/v10/final drift and ambiguous "done vs planned" language.
